Version History (since Version 2.3.0.0)
Version 3.6.2.2 (2008-08-23)
  • Corrected Bug: The "Print"-dialogs (for Sudokus and for booklets) didn't show up on 64-bit systems.
Version 3.6.2.0 (2008-06-29)
  • New feature: It is now possible to delete values using the <Del>-key.
  • Corrected Bug: The application crashed when selecting "Validate".
Version 3.6.1.0 (2007-11-10)
  • New feature: New option "Save State".
  • Corrected Bug: The application crashed when selecting "Reset Problem".
Version 3.6.0.1 (2007-09-30)
  • New feature: The name of the publisher changed from "cp-solutions" to "pi-c-it"
Version 3.6.0.0 (2007-06-29)
  • Corrected Bug: In some rare cases, the program didn't show hints even though there were hints. This has been fixed.
  • Corrected Bug: The menu entry "Problem|Clear Candidates" was enabled even though there were no manually entered candidates. This has been fixed.
Version 3.5.3.0 (2007-05-17)
  • New feature: New option to clear all manually entered candidates ("Problem|Clear Candidates").
  • New feature: Some layout changes.
Version 3.5.2.0 (2007-05-07)
  • New feature: The constrast of the "X" in the printout of X-Sudokus wasn't (at least for several printers) bright enough. Therefore, a new option has been introduced to adjust the constrast of the "X" in the printout of X-Sudokus.
  • Corrected Bug: The build date in the About-Box wasn't correct. This has been fixed.
Version 3.5.1.0 (2007-04-29)
  • New Feature: Cells which contain an invalid value are marked with a red exclamation mark and an appropriate error message.
  • Corrected Bug: If the user enters an invalid value in a cell and then clicks into another cell, the application crashed. This has been fixed.
Version 3.5.0.0 (2007-04-28)
  • New Feature: A new option, which defines whether or not the severity level of new generated problems can be selected.
  • New Feature: It is now possible to enter values in cells using the mouse wheel.
Version 3.4.2.0 (2007-04-15)
  • New Feature: Enhanced calculation of possible values for the diagonals for X-Sudokus.
Version 3.4.1.0 (2007-04-12)
  • Corrected Bug: Saving a Sudoku problem didn't correctly store the type of problem. This has been fixed.
Version 3.4.0.0 (2007-04-11)
  • New Feature: Support of X-Sudokus.
  • Corrected Bug: In same cases the program crashed if the generation process was interrupted. This has been fixed.
Version 3.3.0.0 (2007-02-04)
  • New Feature: The calculation of valid values for a cell now also identifies "Locked Candidates".
Version 3.2.2.0 (2007-01-19)
  • Corrected Bug: Some internal cleanup/improvement efforts.
Version 3.2.1.1 (2007-01-13)
  • Corrected Bug: The version 3.2.1.0 might be faster than all of the others, but unfortunately under some circumstances it crashes when creating new problems. This has been fixed (without slowing down the application excessively).
Version 3.2.1.0 (2007-01-12)
  • New feature: Some really convincing performance improvements for generating new problems.
Version 3.2.0.0 (2007-01-05)
  • New feature: Besides the "Watch Hand Method" it is now possible to show and print both possible values and candidates as digits.
Version 3.1.2.0 (2007-01-01)
  • New feature: Some performance improvements for generating new problems (unfortunately this function is still not really fast, a redesign might be necessary to calculate new problems much faster).
Version 3.1.1.0 (2006-12-26)
  • New feature: New option to add a comment for the current Sudoku problem. This problem is printed out below the problem.
Version 3.1.0.0 (2006-12-25)
  • New feature: When solving a problem interactively it is now possible to mark "candidates" for cells. When printing, these candidates are printed also.
  • Corrected Bug: When calculating more than one solution for a problem, it happend in some cases that the displayed solution was marked as "invalid" even though it was a perfect solution for the given problem. This has been fixed.
  • Corrected Bug: When prining a problem it sometimes happend that not only the given problem but also some other (calculated) values in some cells were printed out. This has been fixed.
Version 3.0.1.0 (2006-12-09)
  • New feature: The program now shows only those hints, which are quite obviously and not dependent on solving other hints.
  • Corrected Bug: In some cases the program didn't handle modified problems correctly. This has been fixed.
Version 3.0.0.0 (2006-11-25)
  • New feature: The "Track"-option now shows the whole solving process including those cells that are not solved using backtracking.
Please note: Due to internal variations this version isn't compatible with prior versions of Sudoku. If you have already installed an older version of Sudoku you must uninstall the older one prior to installing the current one (using the regular "Add or Remove Program" control panel of Windows XP). The automatic update will not update older versions of Sudoku to the current one!
Version 2.7.2.1 (2006-11-12)
  • Corrected Bug: The option "Hide when minimized" didn't work as expected. This had been fixed.
Version 2.7.2.0 (2006-10-29)
  • New feature: The program now displays hints in different colors (red for obvious cells, orange for cells harder to detect).
  • New feature: Some performance improvements.
Version 2.7.1.0 (2006-10-08)
  • New feature: Added option "Hide when minimized".
  • New feature: Acoustic hint (simple beep) when the entered value leads to an unresolvable problem (only if option "Auto Check" is checked).
Version 2.7.0.0 (2006-10-01)
  • New feature: The elapsed solving time for a problem is now saved and reloaded with the problem.
  • New feature: The solving time can be reset using the appropriate menu item.
  • New feature: The counting of the elapsed solving time can paused.
Version 2.6.3.1 (2006-09-09)
  • Corrected Bug: After saving a problem the dirty flag wasn't reset. This has been fixed.
Version 2.6.3.0 (2006-08-27)
  • New feature: The problem info now also show the internal severity level.
  • Corrected Bug: If the directory from which the problems for a booklet had to be loaded, was empty, the application crashed. This has been fixed.
Version 2.6.2.1 (2006-08-14; Version 2.6.2.0 has never been published)
  • New feature: It is now possible to interrupt the creation of a booklet out of existing problems.
  • New feature: New-and much better suitable-icon for the Sudoku application.
  • New feature: Link to the Sudoku-Homepage from the "Help"-menu.
  • Corrected Bug: In very rare cases the application crashed on MP systems while generating new problems. This has been fixed.
Version 2.6.1.2 (2006-08-11)
  • Corrected Bug: In some cases it was not possible to cancel the generation of a new problem. This has been fixed.
Version 2.6.1.1 (2006-08-09)
  • New feature: The picture in the "About Box" has been exchanged.
  • Corrected Bug: The solution is not longer displayed on the screen when a problem is printed out.
  • Corrected Bug: The application crashed if the user cancelled the print output. This has been corrected.
Version 2.6.1.0 (2006-08-08)
  • New feature: The performance when creating a booklet out of existing problems has been increased.
  • Corrected Bug: In some cases the menu item "Show Possible Values" was enabled, even though this function wasn't actually allowed.
Version 2.6.0.0 (2006-08-06)
  • New feature: The possibility to set the severity level of generated problems was introduced.
  • New feature: The check for unsaved changes now takes place in any case the current problem is reset.
Version 2.5.0.1 (2006-08-04)
  • New feature: If the user closes the application and there are unsaved changes in the current problem, the user is now asked if she wants to save the problem.
  • Corrected Bug: The solving time was not displayed in case of the "Auto Check"-option was selected. This has been fixed.
Version 2.5.0.0 (2006-08-02)
  • New feature: The number of problems for booklet out of existing problems can now be set.
  • New feature: The time used for solving a problem interactively is now been displayed.
Version 2.4.2.0 (2006-07-21)
  • New feature: Some changes regarding options. In addition the setting of the options "Track", "Find all Solutions", and "Auto Check" are now saved when closing the program.
Version 2.4.1.0 (2006-04-09)
  • New feature: The calculation of the severity level has changed. In most cases the resulting level is the same as before, however in some times the new level better reflects the "real" severity (if there is a such a thing like a "real" severity).
  • New feature: Performance enhancements in particular when creating a booklet from stored problems.
Version 2.4.0.0 (2006-03-26)

Version 2.3.3.0 (2006-03-12)
